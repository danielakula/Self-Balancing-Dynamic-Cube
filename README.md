# Self-Balancing-Dynamic-Cube
Cubli inspired project for 3rd year undergraduate dissertation. Objective is to achieve balance on three edges, corner, and jump from face to edge.


# Contents
ESP32_Tests is written code for each tests done on the firmware.

Edge Balance is the code that implements an inner LQR control loop and a PI outer loop to achieve edge balance on Motor1.

UI Balance is a test which adds wireless UI onto Edge Balance.
